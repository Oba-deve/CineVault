# CineVault - Movie Search & Watchlist

A dark-themed movie discovery app inspired by Netflix. Search for movies, view details, and build your personal watchlist.

<img width="1431" height="813" alt="Screenshot 2026-07-07 at 11 19 04 PM" src="https://github.com/user-attachments/assets/d71b27b1-833b-445b-98d4-f0c0dbe37008" />

## ✨ Features

- Search movies by title using TMDB API
- Beautiful responsive movie cards with posters
- Detailed movie modal (plot, cast, rating, runtime)
- Add or remove movies from personal Watchlist
- Persistent Watchlist using `localStorage`
- Dark cinematic UI with Netflix-style red accents
- Fully responsive design (mobile + desktop)

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- TMDB API (for real movie data and posters)
